# 神奇的数字3
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
在计算机的世界里，0和1总是很神奇的数字，不过，在双十一这个特殊的日子里，3也是一个神奇的数字，比如三角⋯⋯咳，下面给定一个只由0和1组成的整数，请判断这个数能否被3整除。


## Input
每一行包括一个由0和1组成的长度不超过1000的十进制整数

读至文件尾

## Output
每个输入对应一个输出，每个输出占一行，如果输入的整数可以被3整除，输出YES，否则输出NO

## Sample Input
```
010

```
## Sample Output
```
NO

```
